# FishTVZB
小马哥 斗鱼直播模仿

￥￥ 项目前准备： 获取其他 App 资源包 方法
    1. 下载  .ipa Application 安装包
    2. 解压安装包，右键进入 .app 资源包， 获得 icon,launch 图片资源
    3. 其他大部分资源封装在 Assets.car 文件
    4. 利用 Cartool 工具解压 Assets.car 文件
       --. 进入 edit scheme 设置界面
       --. 把Assets.car 文件地址导入到 Argument 
       --. 再把 解压后保存资源的目标文件夹地址导入到 Argument 
       --. 运行即可
